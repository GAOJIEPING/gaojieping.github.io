title ：搭建hexo个人博客

简介

目前，互联网资源的普及，已经可以让人人搭建一个个人博客了；这边就教大家怎么搭建Hexo并配置。大家可以参考Hexo官方文档配合使用。持续更新～

前端

widows端

安装：

  1. Node.js
  2. Git
  3. 文本编辑器（vim,vsCode等）

Windows创建Hexo目录，再简单不过了；只需要在你知道的地方：右键鼠标-新建文件夹-重命名文件夹：

部署Hexo本地服务

使用Npm安装Hexo

因为Node里自带Npm，所以我们可以直接使用Npm；这一步，总的来说，就是：npm install -g hexo；为了方便国内用户，提前加了一步更换npm为腾讯云源，非大陆用户可以跳过，Windows用户打开cmd或powershell输入，Mac和Linux用户可以直接在Terminal内输入：

npm config set registry http://mirrors.cloud.tencent.com/npm/
Windows

在安装Node.js和Git以后，Windows用户打开cmd或powershell输入： 

npm install -g hexo

初始化Hexo：

hexo init

构建Hexo初始化Hexo后，其实Hexo就可以使用了；

构建Hexo：

hexo g

会在当前目录下生成一个叫public的文件夹，这个文件夹就是根据你Hexo的配置，生成的HTML+CSS+JS的静态文件了。启动Hexo内建服务器其实，整个public文件就是一个博客了，但是我们还要部署Nginx等网络环境或者使用HBuilderX等根据，启动Web服务器才能用浏览器本地IP(即：127.0.0.1或localhost)加端口进行访问；为了减少麻烦，可以使用Hexo内建服务器：

hexo s

如果出现未知命令，先执行：

npm install hexo-server --save

执行完成后，再执行

hexo s

即可。
完成上述操作，浏览器访问：127.0.0.1:4000或localhost:4000即可访问博客:
